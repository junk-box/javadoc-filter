これは？

javadocのクラス一覧をフィルタリングする ブックマークレット※ です。
フィルタリングにより目的のクラスへ素早くアクセスが可能です。

※ブックマークレット ・・・ ユーザーがウェブブラウザのブックマークなどから起動し、ウェブブラウザで簡単な処理を行う簡易的なプログラムのこと  【Wikipediaより】

インストール

次のリンク先をブックマークとして登録します。

JavaDoc Filter

使用方法

javadocのトップページを開いた状態で、ブックマークをクリックするとJavaDoc Filterが起動します。 ※ ブラウザにより起動に時間がかかります。

フィルター入力エリアが表示されたら、クラス名を入力します。

FAQ

オフラン環境だとブックマークレットが起動しません

本ブックマークレットは起動時にネットワーク経由でライブラリをロードしています。そのため、オフライン環境ではご利用になれません。

Google ChromeでローカルのJavadocに対してブックマークレットが起動しません

Google ChromeではデフォルトでローカルのHTMLから他のURLの読み込みが許可されていません。
以下の引数を起動時に指定することでローカルHTMLにJavaDoc Filterのロードが許可されます。(セキュリティレベルが下がるため、引数の指定は各自の自己責任でお願いします) 

--allow-file-access-from-files

Windowsの場合、ショートカットのプロパティに設定しておくと便利です。
